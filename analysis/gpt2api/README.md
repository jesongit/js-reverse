# gpt2api

本目录用于基于真实网页运行态与网络证据，分析 `https://chatgpt.com` 的对话请求链路，并沉淀面向学习用途的高层分析结论。

## 目标

- 分析对话请求的真实构造方式
- 分析网页模型标识与请求参数映射
- 分析真实流式响应机制
- 分析会话状态维护方式
- 输出安全可分享的分析报告

## 产物

- `WORKLOG.md`：滚动工作记录
- `report.md`：稳定的分析报告

## 约束

- 以真实网络与运行态证据为准
- 仅输出学习用途的高层分析，不提供可直接复用的私有接口调用实现
- 不在文档中记录滚动过程，过程记录统一写入 `WORKLOG.md`
